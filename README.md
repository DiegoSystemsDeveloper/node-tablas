#Notas
Programa que realiza tablas de multiplicar

```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Toma la base de la tabla de multiplicar    [número] [requerido]
  -l, --listar   Lista la tabla de multiplicar
                                         [booleano] [requerido] [defecto: false]
  -h, --hasta    Toma hasta donde va la tabla de multiplicar            [número]
```
